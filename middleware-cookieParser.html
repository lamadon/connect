cookieParser

lib/middleware/cookieParser.js

Parse Cookie header and populate req.cookies with an object keyed by the cookie names. Optionally you may enabled signed cookie support by passing a secret string, which assigns req.secret so it may be used by other middleware such as session().

Examples

connect.createServer(
    connect.cookieParser('keyboard cat')
  , function(req, res, next){
    res.end(JSON.stringify(req.cookies));
  }
);
  • param String secret
  • returns Function
